SSH本地端口转发

通俗一点: 内网穿透

A <===> B <==|NAT/防火墙|==> C

由 C 发起，将 C 本机或局域网内的端口映射到 B，A 可通过 B 访问到 C

1

ssh -gN -R $ListenIP:$ListenPort:$LocalIP:$LocalPort $RemoteUser@$RemoteIP
  • $ListenIP: B 要监听的 IP
  • $ListenPort: B 要监听的端口
  • $LocalIP: C 本地的目标 IP
  • $LocalPort C 本地的目标端口
  • $RemoteUser B 上的用户
  • $RemoteIP B 的 IP

比如将本机的 Minecraft 服务器映射到 172.16.0.1 (用户名: User)

1

ssh -gN -R 0.0.0.0:25565:127.0.0.1:25565 User@17.16.0.1

注意

B 机上的 ssh 需开启GatewayPorts

#!/bin/sh /etc/rc.common

START=99
USE_PROCD=1

BINARY=/usr/bin/frps
CONF=/etc/frps/frps.ini

start_service() {
        procd_open_instance
        procd_set_param command $BINARY -c $CONF
        procd_set_param file $CONF
        procd_set_param respawn
        procd_close_instance
}
